## GitHub Repositories

- [AweSim (Awesome Simulations)](https://github.com/SINTEF/thermopack) — pip-installable Python API (thermopack). Direct NumPy integration for property arrays. MIT license allows unrestricted use. Can replace proprietary PVT packages in assetutilities.

- [DWSIM](https://github.com/DanWBR/dwsim) — .NET/Mono based; limited direct Python integration. Can be driven via COM automation or CLI. Consider coupling with assetutilities for property calculations. Thermodynamic packages useful as reference data.

- [Fiona](https://github.com/Toblerity/Fiona) — Pythonic wrapper around OGR. Used as I/O backend by GeoPandas. Useful for streaming large spatial datasets that don't fit in memory.

- [Fluids](https://github.com/CalebBell/fluids) — pip-installable, pure Python. Works with thermo library for fluid properties. Comprehensive correlation library that can validate digitalmodel's pipeline calculation modules.

- [Thermo](https://github.com/CalebBell/thermo) — pip-installable, pure Python. Companion library fluids provides pipe flow calculations. Direct integration with pandas DataFrames for bulk property lookups.
