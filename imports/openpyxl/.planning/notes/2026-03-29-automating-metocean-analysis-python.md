Automating Metocean Analysis with Python

Over the past period, I've developed a fully automated Python-based workflow for metocean studies, aimed at transforming raw environmental data into clear, engineering-ready insights.

Using libraries such as pandas, numpy, matplotlib, plotly, scipy.stats, and Windrose, the system generates a comprehensive suite of analyses and visualizations

Wind Analysis:
All-years wind rose
Yearly wind roses
Seasonal wind roses (Spring, Summer, Autumn, Winter)
Annual wind speed histogram

Wave Analysis:
All-years wave rose
Yearly wave roses
Seasonal wave roses
Annual wave histogram
Significant wave height histogram

Joint Metocean Relationships:
Wind direction vs wave direction
Wind speed vs significant wave height
Wave period vs significant wave height

Extreme Value Insights:
Top 10 maximum wind speeds (overall & yearly)
Top 10 significant wave heights (overall & yearly)

The goal of this tool is to standardize metocean assessments, reduce manual effort, and ensure consistent, high-quality outputs for engineering applications.

This workflow is particularly useful for:
Offshore & coastal design
Port and harbor studies
Early-stage feasibility assessments

Some of the generated outputs are included in this post.
