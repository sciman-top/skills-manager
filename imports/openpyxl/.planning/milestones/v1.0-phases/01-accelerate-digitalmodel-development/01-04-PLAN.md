  <behavior>
    Core functions (all referencing DNV-RP-C203 Appendix C/D methodology):

    1. `SeaStateEntry` dataclass:
       - Fields: hs_m (significant wave height), tp_s (peak period), probability (0-1), duration_hours (optional, defaults to hours in design life * probability)
       - Validation: probability >= 0, hs_m > 0, tp_s > 0

    2. `ScatterTable` class or list of SeaStateEntry:
       - Probabilities must sum to ~1.0 (within tolerance 1e-3)
       - Helper: `from_arrays(hs_values, tp_values, probabilities)` class method

    3. `scatter_fatigue_damage(scatter_table, stress_transfer_function, sn_curve, method, design_life_years, frequencies) -> ScatterFatigueResult`:
       - DNV-RP-C203 Appendix C/D approach
       - For each sea state in scatter_table:
         a. Generate wave spectrum S(f) for the (Hs, Tp) pair (use JONSWAP from hydrodynamics/wave_spectra.py or accept pre-computed PSD)
         b. Compute response PSD via transfer function: S_response(f) = |H(f)|^2 * S_wave(f)
         c. Compute spectral moments (m0, m2, m4) of response PSD
         d. Use Dirlik (or specified method) to get damage rate from existing frequency_domain module
         e. Compute damage for this sea state: D_i = damage_rate * duration_i
       - Total damage: D_total = sum(D_i) for all sea states
       - Returns ScatterFatigueResult with total_damage, per_sea_state_damage list, fatigue_life_years

    4. `ScatterFatigueResult` dataclass:
       - total_damage: float (Miner's sum)
       - fatigue_life_years: float (design_life / total_damage)
       - per_sea_state: list of dicts with {hs, tp, probability, damage, damage_fraction}
       - method: str
       - n_sea_states: int

    Test cases:
    - Single sea state -> damage matches direct frequency_domain call
    - Two sea states with equal probability -> damage = 0.5 * D1 + 0.5 * D2
    - Probabilities not summing to 1.0 -> raises ValueError
    - Zero Hs sea state -> zero damage contribution
    - All-zero transfer function -> zero total damage
    - Fatigue life = design_life / total_damage

    Use synthetic/simple inputs for tests (not requiring actual standard data):
    - Simple flat transfer function H(f) = constant
    - Known JONSWAP spectrum parameters
    - Simple bilinear SN curve for predictable damage

    ```python
    Computes cumulative fatigue damage by iterating over a scatter diagram
    of sea states, computing spectral fatigue damage for each using existing
    frequency domain methods (Dirlik, Tovo-Benasciutti), and summing per
    Miner's rule.
    """
    from __future__ import annotations
    import numpy as np
    from dataclasses import dataclass, field
    from typing import Callable, Literal, Sequence

    from .frequency_domain import (FrequencyDomainAnalyzer, SpectralMoments,
                                    FrequencyDomainResult)
    from .sn_curves import SNCurveBase

    @dataclass
    class SeaStateEntry:
        hs_m: float
        tp_s: float
        probability: float
        duration_hours: float = 0.0  # computed from design life if 0

    @dataclass
    class ScatterFatigueResult:
        total_damage: float
        fatigue_life_years: float
        per_sea_state: list[dict]
        method: str
        n_sea_states: int
        design_life_years: float

    def scatter_fatigue_damage(
        scatter_table: Sequence[SeaStateEntry],
        stress_transfer_function: Callable[[np.ndarray], np.ndarray] | np.ndarray,
        sn_curve: SNCurveBase,
        frequencies: np.ndarray,
        method: Literal["dirlik", "tovo-benasciutti", "narrow_band"] = "dirlik",
        design_life_years: float = 25.0,
        wave_spectrum_func: Callable | None = None,
    ) -> ScatterFatigueResult:
        ...
    ```
