4. **Executive Summary** (`.highlight-box`):
   - Challenge: A subsea flowline project required wall thickness design simultaneously satisfying operator preference (ASME B31.4), regulatory requirement (API 1111), and third-party verification (DNV-ST-F101). Manual comparison across codes was error-prone and time-consuming.
   - Solution: Automated multi-code wall thickness analysis using digitalmodel's wall thickness modules with systematic comparison across pipe sizes, pressure ratings, and water depths.
   - Result: 40% reduction in design time. Identified that DNV-ST-F101 governed the design for all pipe sizes above 12-inch, while ASME B31.4 was more restrictive for smaller diameters -- a non-obvious finding that influenced procurement.

5. **Project Background section:**
   - Business context: Deepwater flowline connecting subsea manifold to host facility. Operator required ASME B31.4 compliance, regulatory body required API 1111, third-party verifier required DNV-ST-F101 checks. All three had to pass.
   - Technical challenge: Each code handles burst, collapse, and propagation differently -- different safety factors, different formulas, different treatment of corrosion allowance. ASME B31.4 S403.2.1 uses effective wall thickness (t - CA), while DNV-ST-F101 separates material and condition factors.
   - Comparison table: ASME B31.4 (burst: hoop stress ratio, collapse: ovality formula), API 1111 (burst: Barlow with safety factor, collapse: elastic-plastic), DNV-ST-F101 (burst: characteristic resistance, collapse: propagation buckling)

6. **Solution Approach:**
   - Systematic parameter sweep: 5 pipe ODs (8, 10, 12, 14, 16 inch), 3 wall thicknesses per OD, design pressure from 5000 to 10000 psi, water depths 500-2500m
   - Each combination evaluated against all 3 codes simultaneously
   - Governing code identified per pipe size and water depth combination
   - Tools: digitalmodel wall thickness module (Python), aceengineer.com wall thickness calculator (interactive demo)
   - Standards compliance table: ASME B31.4 (S403.2.1 burst, S403.2.2 collapse -- Compliant), API 1111 (Sec 4 design -- Compliant), DNV-ST-F101 (Sec 5 wall thickness -- Compliant)

7. **Results section:**
   - Comparison table: Design Time (4 weeks vs 2.5 weeks, 40% reduction), Code Comparisons (manual: 1 size at a time, automated: all sizes simultaneously), Governing Code Identification (manual: assumed single code, automated: per-size identification), Material Savings ($0 vs $1.2M identified through optimal wall thickness selection)
   - Key finding: DNV-ST-F101 collapse check governed for 14-inch and 16-inch pipe at deepwater locations (>1500m), while ASME B31.4 burst check governed for 8-inch and 10-inch pipe. This cross-over point at 12-inch had not been identified in prior projects using single-code analysis.
   - Impact: Optimal wall thickness selection per pipe size (not uniform over-design) saved $1.2M in steel procurement

8. **Lessons Learned:**
   - What worked: automated multi-code comparison revealed cross-over effects invisible to single-code analysis
   - Challenges: corrosion allowance treatment differs between codes -- ASME deducts from thickness, DNV uses material factors. Consistent input parameters required careful mapping.
   - Recommendations: always compare governing codes across the full range of pipe sizes and conditions; document code-specific assumptions clearly for third-party verification
