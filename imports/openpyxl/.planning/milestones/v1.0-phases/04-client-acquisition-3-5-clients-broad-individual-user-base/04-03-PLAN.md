**Part B: Add scroll depth tracking to both new case study pages**

For each of the 2 new case study HTML files, add the following `<script>` block just before the closing `</body>` tag (after the footer include, before `</body>`):

**pipeline-on-bottom-stability-assessment.html:**
```javascript
<script>
// Scroll depth tracking for case study engagement (Phase 4, D-15)
(function() {
    var thresholds = [25, 50, 75, 100];
    var fired = {};
    var caseName = 'pipeline_obs_assessment';

    window.addEventListener('scroll', function() {
        var docHeight = document.body.scrollHeight - window.innerHeight;
        if (docHeight <= 0) return;
        var scrollPercent = Math.round((window.scrollY / docHeight) * 100);
        for (var i = 0; i < thresholds.length; i++) {
            var threshold = thresholds[i];
            if (scrollPercent >= threshold && !fired[threshold]) {
                fired[threshold] = true;
                if (typeof gtag !== 'undefined') {
                    gtag('event', 'case_study_scroll', {
                        'event_category': 'content',
                        'case_study_name': caseName,
                        'percent_scrolled': threshold
                    });
                }
            }
        }
    });
})();
</script>
```

**multi-code-wall-thickness-comparison.html:**
Same script but with `var caseName = 'multi_code_wall_thickness';`
