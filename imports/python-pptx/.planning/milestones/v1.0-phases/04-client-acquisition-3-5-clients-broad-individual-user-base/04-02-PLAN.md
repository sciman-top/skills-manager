In the existing `<script>` block at the bottom of the page (after the `selectVertical` function definition and the `DOMContentLoaded` listener), add a change event listener for the project type selector. Insert BEFORE the existing form submission tracking code:

```javascript
// Track project type selection
document.getElementById('project-type').addEventListener('change', function() {
    if (this.value && typeof gtag !== 'undefined') {
        gtag('event', 'contact_project_type', {
            'event_category': 'contact',
            'project_type': this.value
        });
    }
});
```

**Change 3: Enhance form submission tracking**

Update the existing `contact_form_submission` event (the `addEventListener('submit', ...)` at the bottom) to also capture the project type. Change from:
```javascript
gtag('event', 'contact_form_submission', {
    'event_category': 'contact',
    'event_label': document.getElementById('subject').value
});
```
To:
```javascript
if (typeof gtag !== 'undefined') {
    gtag('event', 'contact_form_submission', {
        'event_category': 'contact',
        'event_label': document.getElementById('subject').value,
        'project_type': document.getElementById('project-type').value || 'not_selected'
    });
}
```

Note: Add the `typeof gtag` guard that was missing from the original submit handler.

**Change 4: Add project_type to URL parameter handling**

In the existing `DOMContentLoaded` handler, add URL param support for project_type (after the existing `vertical` parameter handling):
```javascript
// Handle project-type parameter
var projectType = urlParams.get('project_type');
if (projectType) {
    var projectTypeSelect = document.getElementById('project-type');
    if (projectTypeSelect) {
        for (var i = 0; i < projectTypeSelect.options.length; i++) {
            if (projectTypeSelect.options[i].value === projectType) {
                projectTypeSelect.value = projectType;
                break;
            }
        }
    }
}
```

Use `var` throughout (not const/let) per established codebase pattern.
