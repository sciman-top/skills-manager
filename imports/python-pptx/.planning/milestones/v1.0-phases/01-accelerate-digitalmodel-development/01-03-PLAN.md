  <behavior>
    ASME B31.4 (Liquid Transportation Systems for Hydrocarbons) design checks:

    1. Burst check (Barlow formula, ASME B31.4 S403.2.1):
       - P_d = (2 * S * t * F * E * T) / D
       - Where: S=SMYS, t=wall_thickness-corrosion_allowance, F=design_factor (0.72 default), E=longitudinal_joint_factor (1.0 for seamless), T=temperature_derating (1.0 for <120C)
       - Utilisation = P_internal / P_d
       - Test: D=0.3048m (12"), t=0.0127m (0.5"), SMYS=358.5MPa (API 5L X52), P_int=10MPa
         -> P_d = 2 * 358.5e6 * 0.0127 * 0.72 * 1.0 * 1.0 / 0.3048 = 21.50 MPa (approx)
         -> Utilisation = 10.0 / 21.50 = 0.465 (approx)

    2. Collapse check (elastic-plastic transition, ASME B31.4 S403.2.2 / API-like):
       - P_el = 2 * E_steel * (t/D)^3 / (1 - nu^2)
       - P_y = 2 * SMYS * t / D
       - P_c via characteristic equation (same as API/DNV approach)
       - Utilisation = P_external / P_c
       - Test: Same pipe, P_ext=5MPa -> utilisation < 1.0

    3. Propagation buckling check (ASME B31.4 S403.2.3):
       - P_pr = 24 * SMYS * (t/D)^2.4 (Battelle/AGA formula)
       - Utilisation = P_external / P_pr
       - Test: Same pipe, P_ext=5MPa -> specific utilisation value

    Edge cases:
    - Zero wall thickness -> division by zero handled gracefully (raise ValueError or return inf utilisation)
    - Corrosion allowance reducing effective thickness to near zero -> high utilisation

    All tests use pytest.approx with abs tolerance documented per Pitfall 2 (intermediate rounding).
  </behavior>
