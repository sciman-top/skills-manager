## Software Ecosystem

No ABAQUS, ANSYS, or OrcaFlex found. The archive uses:
- **2H Offshore Suite** — proprietary riser analysis (KeyGen batch control, TB12XL post-processing)
- **Flexcom-3D** — marine riser FEA (binary .inp model files, .mdl/.inf generated by Excel tools)
- **Shear7** — VIV analysis (structural/hydrodynamic input .dat, modal data common.inp)
- **MATLAB** — post-processing (Shear7 output parsing)
- **Excel + VBA** — engineering calculations and model generation
