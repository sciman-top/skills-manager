# Tax Filing Session Setup with GitHub Tracking

Before opening tax software, establish traceability by: (1) reading all prepared tax documents to build a complete picture; (2) creating a filed-plan checklist with key data points (filing status, dependent info, W-2 totals, prior AGI); (3) opening or referencing existing GitHub issues for the taxpayer repo; (4) posting a session-start comment on the relevant issue with the plan summary and expected numbers. This ensures audit trail and prevents rework if session is interrupted.
