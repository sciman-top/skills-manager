  <behavior>
    Core functions to implement (all from DNV-RP-F109 2021):

    1. `hydrodynamic_force_per_meter(rho_w_kg_m3, D_outer_m, U_m_s, a_m_s2, C_D, C_M) -> float`
       - DNV-RP-F109 S3.2.1, Eq 3.1
       - F_H = 0.5 * rho * C_D * D * |U| * U + rho * C_M * (pi/4) * D^2 * a
       - Zero velocity -> zero force
       - Pure drag (a=0): result = 0.5 * 1025 * 0.9 * 0.5 * 1.0 * 1.0 = 230.625 N/m
       - Pure inertia (U=0): result = 1025 * 3.29 * (pi/4) * 0.25 * 1.0 = 662.18 N/m (approx)

    2. `lift_force_per_meter(rho_w_kg_m3, D_outer_m, U_m_s, C_L) -> float`
       - DNV-RP-F109 S3.2.1, Eq 3.2
       - F_L = 0.5 * rho * C_L * D * U^2
       - Zero velocity -> zero lift
       - Known inputs: rho=1025, C_L=0.9, D=0.5, U=1.0 -> 230.625 N/m

    3. `submerged_weight_per_meter(D_outer_m, t_wall_m, rho_steel_kg_m3, rho_coat_kg_m3, t_coat_m, rho_contents_kg_m3, rho_w_kg_m3) -> float`
       - DNV-RP-F109 S2.2
       - Ws = (W_steel + W_coat + W_contents - W_displaced) * g
       - Must be positive for pipe to stay on bottom

    4. `absolute_stability_check(W_s_N_m, F_H_N_m, F_L_N_m, mu_soil) -> StabilityResult`
       - DNV-RP-F109 S4.3.1, Eq 4.1
       - gamma_SC * (F_H + mu * F_L) / (mu * W_s) <= 1.0
       - Default gamma_SC = 1.1 for normal safety class
       - Returns StabilityResult(utilisation, is_stable, details)

    5. `generalized_stability_check(W_s_N_m, F_H_N_m, F_L_N_m, mu_soil, F_R_N_m) -> StabilityResult`
       - DNV-RP-F109 S4.3.2, Eq 4.5
       - Includes passive soil resistance F_R
       - gamma_SC * F_H / (mu * (W_s - F_L) + F_R) <= 1.0

    Edge cases:
    - Zero submerged weight -> utilisation = infinity -> not stable
    - Negative submerged weight (buoyant pipe) -> not stable
    - Negative velocity -> drag reverses direction (absolute value in drag term)

    Test data: Use independently calculated values with documented equations.
    All tests use `pytest.approx` with documented tolerances.
