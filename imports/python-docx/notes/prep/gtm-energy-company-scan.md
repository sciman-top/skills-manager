## Role Types and Demand Patterns

### High-Demand Specializations (ACE Engineer Core Competencies)

#### 1. OrcaFlex/Mooring & Riser Analysis Engineers
- **Frequency**: Very High (15-20% of relevant postings)
- **Typical Requirements**:
  - 3+ years OrcaFlex experience for dynamic analysis
  - Proficiency in riser design (flexibles, steel catenary, top tensioned)
  - Mooring system analysis (chain, wire, synthetic, hybrid)
  - Vessel response analysis (RAOs, motion sickness, operability)
  - API RP 2SK, DNV-RP-E305 compliance knowledge
- **Hiring Companies**: 2H Offshore, InterMoor, SOFEC, Aker Solutions, TechnipFMC, Shell, BP
- **Salary Range**: $90,000 - $140,000 base + bonus (consulting premium: +15-25%)

#### 2. FEA & Structural Analysts (ANSYS/Abaqus)
- **Frequency**: High (20-25% of relevant postings)
- **Typical Requirements**:
  - ANSYS Mechanical/AutoDynamic for offshore structures
  - Fatigue analysis (API 579, DNV-RP-C203)
  - Buckling and ultimate strength assessment
  - Joint and connector design (K-joints, Y-joints, grouted connections)
  - Python scripting for automation and parametric studies
- **Hiring Companies**: Stress Engineering, Zentech, Wood Group, Worley, McDermott, DNV, ABS
- **Salary Range**: $85,000 - $130,000 base

#### 3. Cathodic Protection & Corrosion Engineers
- **Frequency**: Medium-High (10-15% of relevant postings)
- **Typical Requirements**:
  - Impressed current cathodic protection (ICCP) system design
  - Sacrificial anode systems for subsea structures
  - Corrosion monitoring and inspection techniques
  - ISO 15589, DNV-RP-B401, NACE standards knowledge
  - Field experience with ROV surveys and potential measurements
- **Hiring Companies**: TechnipFMC, Subsea7, Saipem, Wood Group, Worley, BV, LR
- **Salary Range**: $80,000 - $120,000 base

#### 4. API 579/Fitness-for-Service Specialists
- **Frequency**: Medium (8-12% of relevant postings)
- **Typical Requirements**:
  - API 579-1/ASME FFS-1 Level II/III certification preferred
  - Assessment of existing equipment for continued service
  - Damage mechanism identification (corrosion, cracking, deformation)
  - Remaining life calculations and inspection planning
  - Integration with RBI (Risk-Based Inspection) programs
- **Hiring Companies**: Zentech, Stress Engineering, Wood Group, Worley, Operators (major integrateds)
- **Salary Range**: $95,000 - $145,000 base (certification premium)

#### 5. Python Automation & Data Analysts
- **Frequency**: Growing Rapidly (12-18% of relevant postings)
- **Typical Requirements**:
  - Python 3.x with pandas, NumPy, SciPy for data analysis
  - Automation of repetitive engineering tasks (report generation, QA/QC)
  - API integration for metocean data retrieval (NOAA, ECMWF)
  - Development of internal tools and dashboards
  - Experience with Jupyter notebooks and version control (Git)
- **Hiring Companies**: All majors - particularly operators and consultancies building digital capabilities
- **Salary Range**: $85,000 - $135,000 base (premium for domain knowledge + coding)
