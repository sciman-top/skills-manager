# Tax Filing Session Setup with GitHub Traceability

Before opening tax software, read all prepared files (W-2s, dependent info, prior AGI, etc.) to build a complete picture. Create a GitHub issue for the filing session with a summary of expected numbers and key data points. Use the issue as a single source of truth for the session plan, cross-referencing related issues (e.g., payment tracking). This ensures traceability, prevents data re-entry, and creates an audit trail for multi-step tax workflows.
