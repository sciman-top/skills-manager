# Multi-File Tax Prep Orchestration

When handling complex tax returns with multiple source documents, begin by reading all prepared files to build a complete picture before starting data entry. Create a numbered filing plan (10+ bullets) documenting personal info, dependents, income sources, withholding, and expected totals. Use GitHub issues for session traceability—post a comment on the main filing issue with the complete plan and expected AGI/totals for verification against final return. This creates an audit trail and prevents data loss mid-session.
