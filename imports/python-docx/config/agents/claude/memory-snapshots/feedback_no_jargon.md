Do not use jargon abbreviations like TL;DR, LGTM, AFAIK, etc.

**Why:** User asked what TL;DR means — these abbreviations are not universally known and add friction.

**How to apply:** Use plain language summaries. Say "In short:" or just write concisely without a label.
