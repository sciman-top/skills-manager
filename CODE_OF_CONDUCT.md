# Code of Conduct

[English](#english) | [中文](#中文)

## 中文

为保持协作效率与社区健康，请遵循以下原则：

1. 尊重事实，聚焦问题本身。
2. 讨论方案时保持具体、可验证、可复现。
3. 尊重他人，避免人身攻击、歧视或贬损性表达。
4. 骚扰、恶意挑衅、歧视性行为均不可接受。

### Enforcement

维护者可根据情节采取措施，包括但不限于：

- 口头或书面警告
- 关闭讨论、issue 或 PR
- 限制或移除协作权限

### Scope

本准则适用于本仓库相关的 issue、PR、讨论区及其他公开协作场景。

## English

To keep collaboration effective and healthy, please follow these principles:

1. Stay factual and focus on the problem.
2. Keep technical discussion specific, verifiable, and reproducible.
3. Treat others respectfully; personal attacks, harassment, discrimination, and degrading language are unacceptable.
4. Malicious or disruptive behavior is not allowed.

### Enforcement

Maintainers may respond to violations with actions including:

- verbal or written warnings
- closing discussions, issues, or pull requests
- restricting or removing collaboration access

### Scope

This policy applies to issues, pull requests, discussions, and other public collaboration spaces related to this repository.
